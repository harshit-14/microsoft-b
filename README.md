# microsoft-teams-clone
## Working live at :-  https://ms-team-hk.herokuapp.com

### Developed with MERN Stack , Socket.io , Peer.js

## Installation 
-> Backend
```
npm init
```
```
npm install
```


-> Frontend
~~~
cd frontend
~~~
```
npm install
```
```
npm start
```
